# What type of laser pointer can I use with Cozmo?

When Laser chaser is activated, Cozmo can track any standard red or green laser. He likes bright dots!

![Laser_Chaser.png](//d33v4339jhl8k0.cloudfront.net/docs/assets/5e3f0b1e2c7d3a7e9ae777f5/images/5e7d772604286364bc975c76/img-3326-1585280666-1809287804.png)

LED Pointers that project big shapes, such as hearts or moons, will not work.

For safety, DDL recommends FDA Approved, Class 1 lasers. Always use lasers with adult supervision.

Laser Chaser first appeared in the Cozmo 2.1.0 update.
