# What is the difference between Cozmo and Vector?

![](//d33v4339jhl8k0.cloudfront.net/docs/assets/5e3f0b1e2c7d3a7e9ae777f5/images/5e85629c2c7d3a7e9aea3b18/file-hY3MtoDAen.png)

Vector and Cozmo both have a place in the home, but they have different ways to interact with you and your household.

|  |  |
| --- | --- |
| Cozmo | Vector |
| With a big brain and even bigger personality, Cozmo is the ultimate robot for kids to play and learn with.    Ages: 8+ | The home robot that hangs out and helps out!    Ages: 18+ |
| Buy Cozmo if…       - You want a toy - You're about having fun and learning to code in short play sessions - You're looking for a STEM (Science Technology Engineering and Mathematics) and Education product - Your robot is focused around games, nurturing, unlocking skills, and learning to code | Buy Vector if…       - You want to a robot to respond to your commands (instead of app commands) - You want your robot to help you around the house, like set timers and check the weather - You want to get football scores - You want your robot to be able to find his way back to his Charger, always powered on and connected |
| Key Differences: Cozmo       - App-controlled via a compatible Android, Fire OS, or iOS smart device - Education applications including SCRATCH programming via Anki's Code Lab - Interactive games - 3 Cubes | Key Differences: Vector       - Voice command control, your compatible smart device just needed for setup - Autonomous and self-charging - Cloud-connected - 1 Cube - Can take pictures at your command |
| Included in the Box:       - Cozmo Robot - Charging platform - 3 Cubes - Quick Start Guide - Product Information Guide - Cozmo Sticker | Included in the Box:       - Vector Robot - Vector Charger - 1 Cube - Quick Start Guide - Product Information Guide |
| Required / Not Included:       - A current Android, Fire OS, or iOS smart device (see compatible device list) - USB power source (minimum DC Output = 5V/1.5A) | Required / Not Included:       - A current Android, Fire OS, or iOS smart device (see compatible device list) - USB power source (minimum DC Output = 5V/1A) - A validated, free Anki Account |
