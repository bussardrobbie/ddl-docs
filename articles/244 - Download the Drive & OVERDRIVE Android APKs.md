## Installation Instructions

1. Download the APK package for each app that you would like to install.
2. Open your mobile device's file manager. In the Downloads folder (or wherever the app was saved after download), long press on the zipped package file name. A dialog box should appear.
3. Click "Unzip" or "Extract" to unzip the APK package into a new folder.
4. Navigate to the new folder. Click on the .apk file to open an installation dialog box. Click "Install". **Do not launch the app yet.**
   1. If you receive a security error, please go into the Settings app of your mobile device.
   2. Find the Security section.
   3. Look for an option to be able to "Install apps from unknown sources." Enable this option.
   4. Go back to the beginning of Step 4 and reattempt installation.
5. In your File Manager, navigate back to the folder with the APK you just installed.
6. Use the Copy or Move function in your device to select and move/copy both of the ".obb" files into the following directory in your phone's Internal Storage: "Android/obb/<app package name>"
   1. For DRIVE, this will be "Android/obb/com.anki.drive"
   2. For OVERDRIVE, this will be "Android/obb/com.anki.overdrive"
   3. For OVERDRIVE: Fast & Furious Edition, this will be "Android/obb/com.anki.foxtrot"
7. You may now launch the app and start the race!
