# The 8 Basic Tracks

There are 8 Basic Tracks that you can build with OVERDRIVE or the Fast & Furious Edition right out of the box. Knowing the names of these tracks and how to build them can help you to earn Trophies and complete Goals.

For more specifics about track building and individual track pieces, please read **[How to build the best tracks](//support.digitaldreamlabs.com/article/272-how-to-build-the-best-tracks)**.
