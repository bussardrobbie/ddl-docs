# Using the Terminal on Escape Pod

![](//d33v4339jhl8k0.cloudfront.net/docs/assets/5e3f0b1e2c7d3a7e9ae777f5/images/606382df0ad05379b10d2586/file-8XlH7IPCQT.gif)

The Escape Pod offers several views during Onboarding that allow you to use the visual interface, a command line interface, or both!

**Visual Interface:** The standard user experience. Great for easily and quickly working through the Escape Pod onboarding process.
 **Command Line:** Great for fine tuning the onboarding process or using built-in tools to dive deep into configuration.

Please see the [Command Line: Commands & Options](//support.digitaldreamlabs.com/article/374-escape-pod-command-line-commands) article for a complete list of command line tools available in the Escape Pod!

#### Key Combination Table

| Key Combination | Desired View |
| --- | --- |
| Alt + 1 | Visual Interface Only |
| Alt + 2 | Combination Visual Interface (On Left) & Command Line (On Right) |
| Alt + 3 | Command Line Only |
